# RCS1051: Parenthesize condition of conditional expression

| Property | Value   |
| -------- | ------- |
| Id       | RCS1051 |
| Category | Style   |
| Severity | None    |

## Example

### Code with Diagnostic

```csharp
x = y != null ? "true" : "false"
```

### Code with Fix

```csharp
x = (y != null) ? "true" : "false"
```
